# didact

Integrates PaperQA with a cloud SQL database currently based on Supabase. 

## Setup

```
$ pip install -r requirements.txt
```

Create a `.env` file with the following variables:

```
OPENAI_API_KEY=
GEMINI_API_KEY=
ANTHROPIC_API_KEY=
CROSSREF_MAILTO=
SUPABASE_URL=
SUPABASE_SERVICE_KEY=
```
